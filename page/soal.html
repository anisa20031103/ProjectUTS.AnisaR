SOAL PERSAMAAN LINEAR SATU VARIABEL

Agar lebih memahami materi Persamaan Linear Satu Variabel silahkan perhatikan soal dibawah ini

1. Persamaan dari 3x - 2 = 4 adalah ?

Penyelesaian:

3x - 2 = 4

3x = 4 + 2

3x = 6

x = 6/3 = 2

2. Berapa nilai variabel dari 3(x-1) + x = -x + 7 ?

Penyelesaian:

3(x-1) + x = -x + 7

3x - 3 + x = -x + 7

4x - 3 = -x + 7

4x + x = 7 + 3

5x = 10

x = 10/5

x = 2
